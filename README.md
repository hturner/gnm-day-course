# gnm Day Course

Course materials for a one day introduction to generalized nonlinear models 
using the **gnm** and **logmult** R packages.

The materials were updated for
[Biometrics in the Bush Capital 2025](https://biometricsociety.org.au/conference2025/workshop.html),
2025-14-11.

This course was originally prepared for the
[Zurich R Courses series](http://www.zhrcourses.uzh.ch/en/fruehere-kurse/programm/datenanalyse/intro-nonlinear.html), 
2015-05-11.
